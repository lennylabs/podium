{% comment %} Theme toggle plumbing. The button HTML lives in _includes/header_custom.html. This file carries three pieces: 1. A synchronous stylesheet-href swap. The browser blocks paint on the just-the-docs stylesheet load; if we swap the link's href to the dark scheme before that fetch completes, the browser fetches the dark CSS instead and never paints with the light scheme. This eliminates the FOUC for returning visitors whose saved preference is dark. 2. A body.podium-dark class flip. just-the-docs switches themes by swapping stylesheets, not by adding a body class, so any CSS we want to ride along with the theme (the toggle button's own colors) keys off a class we maintain ourselves. 3. The styles for the toggle button. The button is an inline-flex element styled to live as a natural child of .main-header alongside the search box and aux nav. {% endcomment %}
