{% comment %} Theme toggle plumbing. The button HTML lives in _includes/header_custom.html. This file carries two pieces: 1. An early-init script that reads the persisted preference from localStorage and applies it as soon as just-the-docs's jtd object is available, so a returning visitor lands in their chosen scheme instead of the build-time default (light). 2. The styles for the toggle button itself, scoped to the .theme-toggle class. The toggle calls jtd.setTheme(name) (just-the-docs' runtime API) and persists the choice under the localStorage key "podium-theme". See https://just-the-docs.com/docs/customization/ for the API. {% endcomment %}
